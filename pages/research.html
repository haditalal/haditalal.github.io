Previous Academic Projects

This page presents a brief overview of my previous academic projects, including my Master’s thesis and Bachelor’s graduation project.

Master’s Thesis

Building a System for Forecasting the Electrical Load Demand in Baghdad

This project focused on forecasting electrical load demand using eight years of hourly data. Time-series models (SARIMA and transfer function) and neural network approaches (MLP and NARX) were implemented using MATLAB and C#. The evaluation showed that the NARX model achieved the best weekly forecasting performance.

Bachelor’s Graduation Project

Iris Centroid Recognition Using Hough Transform Analysis

A MATLAB-based image processing system was developed to detect iris centroids using Hough Transform techniques, demonstrating accurate identification of circular iris structures.
